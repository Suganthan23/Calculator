# React Calculator

A modern, responsive calculator built with React, Vite, and Tailwind CSS. Features a clean monochrome interface with full keyboard support and real calculator functionality.

## 🌐 Live Demo

**[View Live Calculator](https://suganthan23.github.io/Calculator)**

## ✨ Features

- ✅ **Basic Operations**: Addition, subtraction, multiplication, division
- ✅ **Smart Percentage**: Calculate percentages (50% of 40 = 20)
- ✅ **Expression Display**: Shows current calculation in progress
- ✅ **Keyboard Support**: Full keyboard navigation and input
- ✅ **Real Calculator Behavior**: Functions like Windows Calculator
- ✅ **Chained Operations**: Seamless operation chaining (5 + 3 * 2)
- ✅ **Modern UI**: Clean monochrome design with smooth animations
- ✅ **Responsive Design**: Fixed width, professional layout

## 🛠️ Technologies Used

- **React 18** - Frontend framework
- **Vite** - Build tool and development server  
- **Tailwind CSS** - Utility-first CSS framework
- **JavaScript ES6+** - Modern JavaScript features
- **Custom Hooks** - Clean state management

## 🚀 Getting Started

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
     git clone https://github.com/Suganthan23/Calculator.git
     cd Calculator
2. **Install dependencies**
     npm install
3. **Start development server**
     npm run dev
4. **Open in browser**
     http://localhost:5173
